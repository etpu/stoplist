Instruction

Pour ajouter un téléphone à stoplist: Stoplist > Ajouter > remplissez les champs requis > Enregistrer.

Attention. La règle s'applique uniquement au numéro de téléphone que vous avez spécifié et uniquement lorsque vous composez ce numéro dans le même format que celui indiqué dans le tableau. Si le médium a plusieurs numéros de téléphone, il est conseillé d'ajouter chacun d'eux à la table (y compris le numéro SIP).
